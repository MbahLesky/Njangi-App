## Project Description

The Njangi Application is a mobile app that enables users to create and manage Njangi groups digitally. Any user can create a group and become its admin. Members can join groups, contribute based on the configured schedule, follow payout cycles, view transaction history, and communicate within the group. Additional admins can also be assigned within a group.

---

## Development Methodology

The project follows the **Software Development Life Cycle (SDLC)** approach.

### Phases
1.  **Planning:** Defining scope, objectives, and feasibility.
2.  **Analysis:** Gathering and documenting detailed requirements (SRS).
3.  **System Design:** Designing architecture, database schema, and UI/UX.
4.  **Development:** Coding the mobile application and backend API.
5.  **Testing:** Functional, security, and performance testing.
6.  **Deployment & Maintenance:** Releasing the MVP and providing ongoing support.
